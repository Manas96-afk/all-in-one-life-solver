🤖 AI Setup

Choose how you want to use Life Solver's AI features

AI Mode (Default - Recommended)
Use our shared Gemini AI for all users. Real AI responses, completely free!
Premium Mode (Your Own API Key)
Use your own Gemini or OpenAI API key for unlimited usage and priority access.
Offline Mode
Use built-in responses only. No AI, but works without internet.

✅ Gemini API (Recommended):

  1. Go to Google AI Studio
  2. Sign in with Google account
  3. Click "Create API Key"
  4. Copy the key (starts with "AIza")
  5. Paste it below

✅ Works directly from browser - No backend needed!

✅ Free tier: 60 requests per minute

Privacy: Your API key is stored locally only.

OpenAI API:

  1. Go to platform.openai.com/api-keys
  2. Sign up or log in
  3. Click "Create new secret key"
  4. Copy the key (starts with "sk-")
  5. Paste it below

⚠️ Requires backend proxy due to browser security

Cost: ~$0.002 per request

Privacy: Your API key is stored locally only.
